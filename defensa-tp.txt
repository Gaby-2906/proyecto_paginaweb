Paleta de colores: use principalmente el color verde para el header y footer con la propiedad background-color.El amarillo para el botón del banner. El naranja para que se posicione con el cursor con a:hover.
Flexbox: se utilizó para la barra de navegación y para el logo con display:flex.Para alinear el texto adentro de la foto en el banner, para mostrar las tarjetas de promoción en fila.Para las reseñas y para el footer.

tienda.js: comienzo definiendo mediante una constante el array de objetos de productos de la tienda. Para los primeros 12 utilizo la api fakeStore.El resto los renderize con JavaScript.
Utilice Async/Await funciones asincrónicas porque las peticiones web tardan tiempo en responder. Para evitar que la página se congele mientras espera los datos, el navegador "espera" en segundo plano.Use combinación de datos de la api mediante .map () con los productos locales. y para agregar las imágenes.
Use try/catch Si la API se llega a caer (falla el try), el programa captura el error en el catch y carga los productos directamente desde nuestro array, asegurando que la página nunca se rompa ni quede en blanco."
Use el renderizado mediante una función para limpiar el contenedor con .innerHTML = "".
.forEach() para recorrer los productos. La función generarFiltros para no permitir elementos duplicados. El LocalStorage para que, si el usuario refresca la página, los productos que eligió no se borren.
