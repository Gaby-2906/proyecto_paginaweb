# 🎢 Proyecto Parque Temático - Entrega Final

¡Bienvenido/a al repositorio de mi proyecto de página web adaptada para un parque de diversiones y entretenimiento! Este sitio web fue desarrollado de forma integral combinando diseño de interfaz de usuario, maquetación estructurada y lógica de programación para ofrecer una experiencia interactiva y fluida.

---

## 🚀 Características del Proyecto

El sitio cuenta con una arquitectura de múltiples páginas (`Multi-Page Application`) completamente conectadas, orientadas a brindar información clara al usuario y una experiencia de simulación de compra.

*   **🏠 Inicio (`index.html`):** Presentación del parque con banners dinámicos, novedades y acceso rápido a las secciones principales.
*   **🛒 Tienda Virtual interactiva (`pages/tienda.html`):** Simulación de compra de pases, combos de comida y merchandising (tazas, peluches, remeras) con fotos del catálogo del parque.
*   **📝 Blog y Novedades (`pages/blog.html`):** Espacio dedicado a artículos informativos, eventos especiales y anuncios del parque.
*   **👥 Acerca de Nosotros (`pages/Acerca-de.html`):** Historia del proyecto, visión y detalles del concepto del parque.
*   **📩 Contacto (`pages/contacto.html`):** Formulario integrado para consultas de los usuarios y canales de atención.

---

## 🛠️ Tecnologías Utilizadas

Para el desarrollo de este sitio web se implementaron las siguientes herramientas y lenguajes del desarrollo frontend estándar:

*   **HTML5:** Estructura semántica de todas las páginas del sitio.
*   **CSS3:** Estilos personalizados, paleta de colores del parque y diseño responsivo utilizando layouts modernos.
*   **JavaScript (ES6+):** Lógica de programación enfocada en la interactividad de la tienda virtual, manejo del DOM para añadir/quitar elementos al carrito y cálculo de totales.
*   **Git & GitHub:** Control de versiones para el despliegue y resguardo del código fuente.

---

## 💻 Funcionalidades Destacadas (JavaScript)

El núcleo dinámico del proyecto se encuentra en el archivo `tienda.js`, el cual maneja de forma eficiente:
1.  **Renderizado Dinámico:** Carga de productos mediante manipulación del DOM.
2.  **Sistema de Carrito:** Capacidad de agregar elementos, actualizar cantidades de pases o productos en tiempo real.
3.  **Cálculo Automatizado:** Procesamiento automático del monto final a abonar según las elecciones del usuario.
